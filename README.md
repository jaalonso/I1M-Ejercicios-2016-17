En este repositorio se encuentran los ejercicios del curso de 
[Informática de 1º del Grado en Matemáticas](http://bit.ly/1WYZ1O9) 
de la Universidad de Sevilla.

Para cada relación de ejercicio se muestra su enunciado, las revisiones de las
soluciones colaborativas escritas por los alumnos (hasta su comentario en
clase) y las soluciones definitivas.
